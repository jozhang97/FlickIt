What is this game?

Flick It is a simple game where users flick shapes into the appropriate corners of the screen. It's casual, fun, and free!

Next

Who built this?

Flick It! was originally developed by Abhishek Mangla, Ashwin Vaidyanathan, Jeffrey Zhang, Rohan Narayan, and Shaili Patel as a project for Mobile Developers of Berkeley. It has since been re-released by the developers themselves as a standalone project.
